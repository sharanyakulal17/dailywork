# Online Quiz Maker

A simple web-based quiz platform that allows users to create and take quizzes with multiple-choice questions. Perfect for practicing Math, Technical, and General Knowledge quizzes.  

---

## Features

- **User Authentication:** Login system to track users.  
- **Quiz Creation:** Users can create quizzes with up to 5 questions per quiz.  
- **Predefined Quiz Titles:** Math Quiz, Technical Quiz, General Knowledge.  
- **Quiz Taking:** Users can attempt available quizzes and get immediate results.  
- **Quiz Results:** Shows score and correct answers after completing a quiz.  
- **Mobile Responsive:** Works well on different devices.
